# Arta
Arta adalah aplikasi manajemen keuangan kelas berbasis web yang dirancang untuk menggantikan pembukuan manual yang rumit dan rawan kesalahan. Dengan pendekatan yang modern, aplikasi ini tidak hanya mencatat angka, tetapi juga membangun budaya transparansi dan partisipasi siswa melalui fitur gamifikasi dan voting interaktif.
